こちらは、スタマチがどんなアプリであったかのご紹介のためのページです。

スタマチの使用技術

  • クライアント(iOS/Android): Flutter
  • アプリケーションサーバ: Django
  • 補助サービス: Firebase(Analytics/Messaging)
  • Webサーバ: Nginx
  • DB: PostgreSQL
  • コンテナ(開発/本番環境): Docker
  • メールサーバ: Postfix/Dovecot
  • HTTPS: Let's Encrypt
  • その他: Bootstrap
